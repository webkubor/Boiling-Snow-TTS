# 设计规范 (Design System)

## 🎨 色彩系统 (Color Palette)
- 主色: 冷青与银灰（影视化、克制）
- 情绪强调色: 暗红（冲突/爆发）、琥珀（回忆/温度）
- 中性色: 深灰到浅灰用于层级与信息密度管理

## 🔤 排版系统 (Typography)
- 中文正文: 优先使用可读性高的无衬线字体
- 标题: 与正文形成 1-2 个字重差，避免过度装饰
- 文案节奏: 长句用于铺陈，短句用于情绪重击

## 📱 布局与交互 (Layout)
- 移动端优先: 360x800 基准
- 骨架屏: 数据加载期间必须显示骨架屏，严禁空白或 0
- 信息分段: 重要操作路径采用分段确认，降低误触与误执行
