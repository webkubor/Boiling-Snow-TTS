# 更新日志 (Changelog)

本项目遵循 [Semantic Versioning](https://semver.org/lang/zh-CN/) 规范。

## [0.1.0] - 2026-02-27

### ✨ 核心特性 (Features)
- **模块化集群架构 (V2)**：重构项目结构，严格隔离克隆、设计、对话、播客四大模式。
- **Apple Silicon 原生加速**：针对 M1/M2/M3 芯片深度优化，引入 MPS 硬件加速与 SDPA 注意力机制。
- **对话剧场模式**：支持多角色台词连续生成，并自动进行场景缝合与呼吸感处理。
- **播客专栏模式**：专为《江湖拾音录》定制，支持自动调音、首尾留白与品牌身份锁定（月栖洲）。
- **AI 样音自动化**：内置 Ref-Opt 组件，支持 1.5s 安全避障裁剪、物理脱水去噪及无损 WAV 格式归一化。
- **智能缓存系统**：基于时间戳的样音缓存，实现秒级重复生成启动。

### 🛠️ 工程优化 (Engineering)
- **JSON 驱动**：文案与逻辑彻底解耦，支持 `movie_config` 与 `podcast_config` 双线运行。
- **自动化版本管理**：新增 `release.sh` 实现一键打包、打标并发布至 GitHub Release。
- **工业级元数据记录**：自动生成角色台词 JSON 记录，实现音频与文案的数字化对齐。
- **多格式支持**：原生兼容 `.m4a`, `.mp3`, `.wav` 参考音频。
