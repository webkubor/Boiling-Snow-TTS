# Boiling-Snow-TTS (沸腾之雪·武侠配音引擎)

本项目是专为微电影《沸腾之雪》量身定制的 **AI 配音生成工具**。它基于阿里巴巴开源的 Qwen3-TTS 技术，通过深度优化和封装，实现了“剧本即配音”的自动化创作流程。

---

## 📖 什么是 TTS？
**TTS (Text-to-Speech)**，即“从文本到语音”。
它是人工智能领域的一项核心技术，能够将任何书面文字转化为自然、流畅的人类语音。
- **声音克隆 (Voice Cloning)**：这是 TTS 的高级进阶版。只需要提供一段 8-10 秒的参考音频（底稿），AI 就能“学习”并完全复刻这个人的音色、语调和呼吸感，从而用这个人的声音说出任何话。

## 🎯 本项目解决了什么问题？
在《沸腾之雪》的创作过程中，我们面临以下痛点，本项目通过 AI 技术将其一一化解：
1. **配音成本高**：无需为每一集寻找专门的配音演员，只需一套角色底稿即可无限生成。
2. **情绪表达难**：通过 **1.7B 全量模型**，支持指令级的情绪控制（如：深沉、肃杀），完美契合武侠剧的张力。
3. **流程繁琐**：通过 **JSON 驱动模式**，实现了文案与代码的彻底解耦。创作者只需改 JSON，AI 自动生成带中文标题的成品。
4. **性能优化**：针对 **Apple M3 Pro (MPS)** 进行了原生硬件加速，让 1.7B 大模型在 Mac 上也能秒级出片。

## ⚔️ 核心功能模块 (Core Modules)

本项目严格区分并隔离了以下两大核心功能，互不干扰：

### 1. 声音克隆 (Voice Cloning) - `Base` 模式
- **逻辑**：AI 学习您提供的 **参考音频** (`assets/reference_audio/`)。
- **触发**：在 `config.json` 中设置 `"model_type": "Base"`。
- **特点**：100% 还原特定人物（如说书人）的音色。

### 2. 音色设计 (Voice Design) - `VoiceDesign` 模式
- **逻辑**：AI 根据您的 **文字描述指令** (`emotion` & `tone`) 凭空捏造声音。
- **触发**：在 `config.json` 中设置 `"model_type": "VoiceDesign"`。
- **特点**：无需任何参考音频，只需描述即可创造独特的江湖角色。

## 🚀 硬件加速优化 (Hardware Acceleration)

本项目核心脚本 `boiling_snow_clone.py` 已针对 **Apple Silicon (M1/M2/M3)** 芯片进行深度原生优化，确保极速生成体验：

- **MPS (Metal Performance Shaders) 加速**：自动探测并启用 Mac GPU 进行矩阵运算，速度远超纯 CPU 模式。
- **SDPA (Scaled Dot Product Attention)**：强制启用原生注意力机制加速，绕过 NVIDIA 专用的 flash-attn 限制，在 Mac 上达到同等加速效果。
- **bfloat16 精度优化**：在 M 芯片上使用 `bfloat16` 数据类型，在大幅降低内存占用的同时，保持了 1.7B 大模型的高音质输出。
- **自动降级保护**：脚本内置环境自检，若 MPS 启动失败将无缝回退至 CPU 模式，确保任务 100% 成功。

## 🤖 模型库详细讲解 (Models Gallery)

项目中 `models/` 目录下包含 4 个配音引擎，按“功能”与“体量”划分为两个梯队：

### 1. Base 家族 (声音克隆引擎 - 核心)
专门用于**复刻/克隆**您提供的参考音频（如说书人音色）。
- **Base-0.6B**：轻量级。速度极快，内存占用低（~2GB），适合创作初期快速生成样音。
- **Base-1.7B**：**全量级 (推荐)**。成品级音质，支持 `emotion` 和 `tone` 指令控制，武侠代入感极强。

### 3. VoiceDesign 家族 (音色设计引擎)
专门用于**凭空创造**全新音色。您只需提供一段文字描述（如“50岁深沉男声”），AI 即可生成对应的独一无二的音色。
- **VoiceDesign-1.7B**：全量设计引擎。支持极其细腻的角色塑造，适合为《沸腾之雪》捏造独特的江湖高手音色。

## 📁 AI 代理执行规范 (SOP)

### 核心操作入口
- **配置文件**：`configs/config.json`（在此修改文案、集数、标题、情绪）。
- **音色库**：`assets/reference_audio/`（存放以 `{persona}_ref.mp3` 命名的 8s 参考音频）。
- **生成结果**：`assets/output_audio/`（自动生成带中文标题的 `.wav` 文件）。

```bash

cd /Users/webkubor/Desktop/create/Qwen3-TTS/
# 执行加速生成脚本
./.venv/bin/python boiling_snow_clone.py
```
